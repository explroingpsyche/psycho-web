### Run Locally

To run **psycho-web.git** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/explroingpsyche/psycho-web.git.git
```

Windows:

```bash
git clone https://github.com/explroingpsyche/psycho-web.git.git
```
